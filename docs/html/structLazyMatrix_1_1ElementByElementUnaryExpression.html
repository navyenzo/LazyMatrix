Detailed Description

template<typename MatrixType>
struct LazyMatrix::ElementByElementUnaryExpression< MatrixType >

A structure to represent element-by-element unary operations on a matrix.

This structure allows for operations like negation, absolute value, square root, etc., to be applied element-wise on a matrix. It uses a functional approach to define the operation that should be applied to each element of the matrix.
