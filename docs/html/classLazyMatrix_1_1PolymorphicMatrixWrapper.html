Detailed Description

template<typename MatrixType>
class LazyMatrix::PolymorphicMatrixWrapper< MatrixType >

Wrapper class that provides a polymorphic interface to a given matrix type.

This class wraps around a specific matrix type and exposes it through the PolymorphicMatrix interface. It handles both const and non-const matrix types, allowing for uniform treatment in a polymorphic context.
