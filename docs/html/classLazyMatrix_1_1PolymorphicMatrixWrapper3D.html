Detailed Description

template<typename MatrixType>
class LazyMatrix::PolymorphicMatrixWrapper3D< MatrixType >

Wrapper class that provides a polymorphic interface to a given 3d-matrix type.

This class wraps around a specific 3d-matrix type and exposes it through the PolymorphicMatrix3D interface. It handles both const and non-const 3d-matrix types, allowing for uniform treatment in a polymorphic context.
