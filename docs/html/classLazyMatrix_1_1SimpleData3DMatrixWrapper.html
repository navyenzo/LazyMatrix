Detailed Description

template<typename SimpleDataType>
class LazyMatrix::SimpleData3DMatrixWrapper< SimpleDataType >

A wrapper class to provide a 3d-matrix-like interface for simple data types.

This class is designed to wrap simple data types such as bool, float, int, etc., giving them a matrix-like interface. It presents these types as 1x1 matrices, allowing them to be used interchangeably with more complex matrix types in operations and functions expecting matrix-like objects.

Template Parameters
SimpleDataTypeThe type of the simple data to be wrapped.
