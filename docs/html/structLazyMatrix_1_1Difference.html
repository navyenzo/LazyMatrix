Detailed Description

template<typename MatrixType>
class LazyMatrix::Difference< MatrixType >

Computes first-order differences of a matrix.

The Difference class is designed to calculate the first-order differences of a given matrix. It supports computing differences either row-wise or column-wise. The class takes in a matrix and the desired direction (rows or columns) for calculating differences. This class extends the functionality of the BaseMatrix class and is templated to work with various matrix types.

Template Parameters
MatrixTypeThe type of the input matrix for which differences are to be calculated.
