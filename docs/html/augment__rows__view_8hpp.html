Detailed Description

Provides functionality to augment two matrices row-wise as a view.

This file contains the AugmentRowsView template class, which is used to create a view combining two matrices by augmenting their rows. This view essentially places the rows of the second matrix below the first matrix, creating a new matrix view without copying the data.
