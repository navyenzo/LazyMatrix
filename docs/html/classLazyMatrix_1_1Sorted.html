Detailed Description

template<typename MatrixType>
class LazyMatrix::Sorted< MatrixType >

Class for sorting rows or columns of a matrix expression using a user specified row or column to use as the sorting vector for all the other rows or columns in the matrix expression.

This one does not allow modification of original matrix data
