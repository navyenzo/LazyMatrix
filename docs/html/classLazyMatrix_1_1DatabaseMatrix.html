Detailed Description

Represents and accesses a SQL database table as a 2D matrix.

This class provides an interface to treat SQL database tables as 2D matrices, enabling matrix-like data access and operations. It uses a DatabaseWindow for caching and efficient data retrieval. The class supports basic matrix operations and ensures safe interaction with the database.
