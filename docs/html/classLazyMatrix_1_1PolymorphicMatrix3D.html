Detailed Description

template<typename DataType>
class LazyMatrix::PolymorphicMatrix3D< DataType >

Base class providing a polymorphic interface for 3d-matrix-like data.

This class defines a generic interface for 3d-matrices, allowing for operations like accessing elements and querying dimensions. It's templated to handle different data types within the 3d-matrix.
