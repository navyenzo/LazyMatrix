Detailed Description

template<typename DataType>
class LazyMatrix::Interval< DataType >

Represents a numerical interval and provides arithmetic operations.

The Interval class template is designed to handle intervals defined by a minimum and maximum value. It provides basic arithmetic operations for intervals and can be used in various mathematical contexts.

Template Parameters
DataTypeThe type of data the interval holds (e.g., int, float, double).
