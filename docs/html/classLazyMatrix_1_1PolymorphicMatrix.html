Detailed Description

template<typename DataType>
class LazyMatrix::PolymorphicMatrix< DataType >

Base class providing a polymorphic interface for matrix-like data.

This class defines a generic interface for matrices, allowing for operations like accessing elements and querying dimensions. It's templated to handle different data types within the matrix.
