Detailed Description

Defines the DatabaseMatrix class and associated safety classes.

This header file provides the definition of the DatabaseMatrix class, which allows accessing and manipulating data in a SQL database table as if it were a 2D matrix. It utilizes Poco's Data library for database interaction and supports basic matrix operations. The file also includes SafeTableName and SafeColumnName classes to prevent SQL injection by sanitizing table and column names.
