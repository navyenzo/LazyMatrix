Detailed Description

Provides functionality for creating views that select specific rows and/or columns from matrices.

This file contains templates for RowSelectorView, ColumnSelectorView, and RowAndColumnSelectorView, which enable users to create views focusing on specific rows, columns, or both from a given matrix. Unlike the non-view selectors, these classes allow modification of the original matrix through the view.
