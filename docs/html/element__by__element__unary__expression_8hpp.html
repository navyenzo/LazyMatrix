Detailed Description

Provides element-by-element unary operations for matrices.

This file is part of the LazyMatrix library. It includes a template structure and functions to perform element-by-element unary operations such as negation, sign, absolute value, square root, and exponential functions on matrices. These operations are essential for various matrix-based calculations and transformations.
