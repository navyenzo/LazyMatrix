Detailed Description

template<typename DataType>
class LazyMatrix::Matrix< DataType >

Template class for a matrix that utilizes memory-mapped files for storage.

This allows the matrix to be shared across multiple processes. It inherits from BaseMatrix using CRTP (Curiously Recurring Template Pattern) to gain a polymorphic behavior while avoiding virtual functions.

Template Parameters
DataTypeThe type of data stored in the matrix.
