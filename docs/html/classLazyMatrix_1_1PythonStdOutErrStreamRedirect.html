Detailed Description

Captures and redirects C++ std::cout/std::cerr and Python's stdout/stderr streams.

This class intercepts and redirects both C++'s std::cout and std::cerr streams as well as Python's stdout and stderr streams. It captures output and error messages generated by both C++ and Python code in a unified manner, storing them in a single sequential buffer. This is useful in scenarios where C++ and Python are used together, such as in applications using pybind11.
