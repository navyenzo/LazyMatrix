Detailed Description

Provides polymorphic wrappers for various 3d-matrix-like data structures in LazyMatrix.

The polymorphic_matrix3d.hpp file defines classes and functions crucial for creating a polymorphic interface for 3d-matrix-like data structures. It allows for uniform handling of different matrix types in a polymorphic context. This file includes the definition of the PolymorphicMatrix3D base class, which offers a generic 3d-matrix interface, and the PolymorphicMatrixWrapper3D class, which wraps around specific 3d-matrix types and exposes them through the PolymorphicMatrix3D interface. These tools are essential for applications where polymorphic behavior with 3d-matrices is required.
