Detailed Description

A mutex class that is robust and suitable for inter-process synchronization.

RobustMutex uses platform-specific implementations to create a mutex that is robust and suitable for use across multiple processes. It ensures that mutexes are not orphaned due to abrupt termination of a process, providing consistent locking behavior.
