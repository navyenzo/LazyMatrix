Detailed Description

template<typename SimpleDataType>
class LazyMatrix::SimpleData2DMatrixWrapper< SimpleDataType >

A wrapper class for simple data types to provide a 2D matrix-like interface.

This class allows simple data types (like int, float, etc.) to be treated as 1x1 matrices. It's particularly useful in a polymorphic context where different data types including matrices and simple data types are handled uniformly.

SimpleDataTypeThe data type of the value to be wrapped.
