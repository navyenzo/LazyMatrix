Detailed Description

Tests for sorting functionality in LazyMatrix.

This file contains test cases to verify the correct implementation of the sorting function in the LazyMatrix library. It tests sorting rows and columns of a 3x3 matrix using the get_sorted_indices function, ensuring robustness and correctness of the sorting.
